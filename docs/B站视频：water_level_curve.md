## 备忘录

圆锥H关于h的隐函数：

$\frac{\pi r^2}{3 h^2} H^3 - \frac{\pi r^2}{h} H^2 + (\pi r^2 - S) H + S c = 0$

抛物线旋转体H关于h的隐函数：

$\frac{\pi r^2}{5 h^4} H^5 - \frac{2 \pi r^2}{3 h^2} H^3 + (\pi r^2 - S) H + S c = 0$

圆锥临界高度：

$h_{\text{crit}} = \frac{S c}{S - \frac{\pi r^2}{3}}$

抛物线旋转体临界高度：

$h_{\text{crit}} = \frac{S c}{S - \frac{8}{15} \pi r^2}$
