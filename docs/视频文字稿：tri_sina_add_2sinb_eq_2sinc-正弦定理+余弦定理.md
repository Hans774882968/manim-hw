我搜到另一道相似但更简单的题
约束方程是一样的，但求的是 cos C
把 cos C 写成余弦定理的形式
然后直接代入约束方程
用一次均值不等式就做完了
这题的思路和那题是一样的
只是目标函数更为隐蔽了

我们先让目标函数出现三角形的边长

由正弦定理，约束方程相当于 a+2b=2c
代入目标函数，消去a

我们发现，f里出现了
b平方加c平方 除以 bc 的结构
这时我们会想：哪里会出现这样的结构呢
余弦定理！
于是我们把 cosA 化简成这种结构

代回f，我们发现f被简化成一个只关于角A的函数

这就是一道高中数学的基本题了
有很多做法，设为k然后辅助角公式可以
几何意义也可以
在此我们采用最简单的几何意义法

式子的主体是单位圆上某一点与定点连线斜率的倒数
注意A是三角形内角，所以只能取上半圆

当连线与单位圆相切，斜率就达到极值
根据勾股定理，最小斜率就是这个数
所以答案是 根号33 除以 3
