辣税人们好
今天主包跟大家从夯到拉评价程序员失业后的路

首先出场的自然是码农们的心头好：外卖员
不仅工作强度可调节、工作时间弹性大
还可以在配送过程中锻炼最短路算法和动态规划能力
而且直到现在工资都不算低
可惜容易被站点、平台逼着闯红灯
码农们跑单时要注意安全！
所以主包只能给它顶级了

接着是服务员
工资低、要加班
一直干上菜、点菜、清洁之类的杂活
而且要面对各种各样的奇葩客人
部分企业还有腻甜企业文化培训
对不善言辞的码农来说属于噩梦
但优点是大部分管吃管住，能保生存，扳回一城
所以主包给到它NPC

接着是保安
保安基本不要求学历，对各位贷款培训
Java、前端才勉强就业的码农们来说很友好
但某些岗位要求形象好，一下筛掉大多数码农
而且要求高强度值班巡逻
并且要处理突发事件和纠纷
比如跟外卖骑手吵架，打架，甚至造成伤亡
不过比较清闲，可以工作时间偷偷刷算法题
准备东山再起，扳回一城
这里给到人上人吧

出路4，世界尽头，进厂
也有两班倒的，但三班倒居多
而且每天基本都是重复同一个动作，非常枯燥乏味
早就被代码弄出各种身体问题的码农很难撑得住
还要小心黑中介克扣工资
不过马内比服务员多一点
这里给到NPC

重量级的来了，创业！
拿惯快钱的码农家庭往往受不了那点小钱

就匆匆上马，开公司开店
叠加目前经济形势
可以说是返贫、家庭破裂最佳途径~
当然极少数能洞察人性、快速响应市场的码农除外哈
